How to use Clash Bot?

All of your scheduling will be done on the Teams page. You can navigate to it using the menu at the top left.

Here are the components and their usage below.

Discord Server Filter For switching between Servers filter_list

You can see which Discord Server you are currently using.

If you want to change, just use the filter button to then select any of your selected guilds.

The 'Free Agents' selection at the top can be used as a open Team for anyone across any server to join.

 New Team Card For creating a brand new Clash Tournament Team add

To start, you can create a brand new Team for one of the upcoming Clash Tournaments in the Discord Server you have selected.

Just click on the plus sign and select the Tournament, Day, and Role you want to sign up with.

 Team Cards For joining existing Clash Tournament Teams login

Each Team contains one of each of the League of Legend's roles.

To add yourself to a Team for a given role, just click on the role and click yes on the dialog box.

To remove yourself from a Team, you can either join another Team or click the minus button next to your name.

Important!

You can only be assigned to a single Team or Tentative Queue per Tournament.

 Tentative Queue For letting others know you are unsure if you are able to play flaky

If you are uncertain if you will be able to play in a Clash Tournament, have no fear! You can use this component to let your Server mates know you are uncertain.

You can use the plus and minus button on the far right of the table to add and remove yourself from the queue.
